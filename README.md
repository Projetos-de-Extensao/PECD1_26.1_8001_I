## Instalação 

Siga os passos abaixo para instalar e executar o projeto localmente.

1. Instale o Node.js (versão recomendada: 18 ou superior) e o npm.
2. Abra o terminal e vá até a pasta do front-end:
   ```bash
   cd src/aac
   ```
3. Instale as dependências do projeto:
   ```bash
   npm install
   ```
4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
5. Abra o endereço exibido no terminal (geralmente `http://localhost:5173`) em seu navegador.

## Uso 

Após iniciar o servidor de desenvolvimento, use o navegador para acessar o app. As alterações nos arquivos são atualizadas automaticamente enquanto o servidor estiver rodando.
