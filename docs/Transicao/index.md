# Tutorial de Uso do Aplicativo

Este documento explica como usar o aplicativo front-end desenvolvido pelo grupo.

## Acesso inicial

1. Abra o navegador web.
2. Navegue até a URL do aplicativo local ou publicado.
3. Na tela de login, informe seu usuário e senha.
4. Caso ainda não tenha conta, utilize a opção de cadastro para criar um novo usuário.

## Navegação principal

Após o login, você terá acesso às principais áreas do sistema:

- **Home / Painel**: contém a visão geral do sistema e caminhos para outras páginas.
- **Cadastro**: formulário para registrar novos usuários ou informações necessárias.
- **Coordenador**: área dedicada às funcionalidades do coordenador do projeto.
- **Secretaria**: seção para as tarefas e dados da secretaria.
- **Notificações**: local onde são exibidas mensagens, avisos e alertas do sistema.
- **Histórico**: página que apresenta registros e informações passadas do sistema.

## Como usar cada página

### Cadastro
- Preencha os campos solicitados no formulário.
- Clique em "Enviar" ou "Cadastrar" para gravar os dados.
- Verifique mensagens de confirmação para garantir que o cadastro foi concluído.

### Coordenador
- Acesse as funções disponíveis para o perfil de coordenador.
- Utilize esta área para consultar ou gerenciar informações de projetos.

### Secretaria
- Navegue pelos formulários e painéis destinados à secretaria.
- Atualize dados e acompanhe os processos administrativos do sistema.

### Notificações
- Consulte as mensagens enviadas pelo sistema.
- Verifique novos alertas ou comunicados importantes.

### Histórico
- Veja registros de ações anteriores.
- Utilize essa página para acompanhar o histórico de operações e eventos do aplicativo.

## Dicas de uso

- Mantenha o navegador atualizado para melhor compatibilidade.
- Sempre faça logout ao finalizar o uso do sistema.
- Em caso de problemas, reinicie o navegador e tente acessar novamente.

## Encerramento

Para sair do aplicativo, finalize a sessão pelo botão de logout (quando disponível) e feche a janela ou guia do navegador.
